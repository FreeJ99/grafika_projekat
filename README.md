# Computer graphics project

This project is a C++ program created using OpenGL, GLFW, GLM and Glad.

The project contains:
* **Models**: dining table, six cakes on the table and three light bulbs
* **Textures**: floor
* **Lighting**: spot light and moving point light, using Blinn-Phong lighting model
* **Shaders**: light shader, object shader and screen shader
* **Anti aliasing**: using a custom framebuffer that maps 2D Multisampled Texture on the screen using a screen shader  
* **By pressing**:
    * **L**: spotlight turns on
    * **E**: grayscale effect turns on
